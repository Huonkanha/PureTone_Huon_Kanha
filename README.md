# PureTone_HuonKanha
## Project Description
PureTone is a modern, responsive landing page for a music streaming service. This project showcases a sleek design with smooth animations, interactive elements, and a mobile-first approach. Built with HTML5, CSS3 (Tailwind CSS), and JavaScript.

 # Features
 ## UI
 - **Modern Dark Theme** with purple and pink gradient accents
 - **Effects** on navigation and cards
 - **Smooth Animations** including fade-in effects and hover transforms
 - **Responsive Design** (Mobile, Tablet, and Desktop)
## Navigation
- **Sticky Navigation Bar** with transparency that becomes solid on scroll
- **Mobile Hamburger Menu** with smooth toggle animation
- **Smooth Scrolling** to different sections
## Content Sections
- **Hero Section** with animated gradient background and pulsing blur effects
- **Albums Section** featuring 6 artist card with hover effects
- **Pricing Plans** with three tiers (Free, Premium Individual, Family)
- **Testimonials (Feedback) Carousel** with automatic rotation and manual controls (Auto-rotate every 5 seconds, Previous/Next navigation buttons, Clickable dot indicators, Smooth fade transitions)
- **Hover Effects** on all interactive elements (Buttons, Cards, Social Icons)
- **Scale Animations** on button clicks and card hovers
## Responsive 
- Mobile-optimized navigation menu
- Flexible grid layouts that adapt to screen size
- Touch-friendly button sizes
- Optimized typography for different viewports

##  How to Run the Project Locally
### Prerequisites
- **Node.js**
### Installation Steps
#### 1. Clone the repository
```bash
git clone https://github.com/huonkanha/PureTone_HuonKanha.git
cd PureTone_HuonKanha
```
#### 2. Install dependencie
```bash
npm install
```
#### 3. Run Tailwind CSS in watch mode
```bash
npm run dev
```
#### 3. Open the project
```bash
Use Live Server in VS Code
```
